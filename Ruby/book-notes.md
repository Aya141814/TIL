## Ruby 超入門

# オブジェクトとは？

Ruby の世界での「もの」
様々なオブジェクトがある
例：文字列オブジェクト、整数オブジェクトなど

# 文字列に計算結果を埋め込むには

#{1+2}と書く！

# また、変数も同じ書き方で埋め込むことができる

```
name = "コアラ"
puts "こんにちは、私は#{name}です"
```

という感じで書ける。

# 変数の名前のルール

- 小文字か、\_で始まるようにする
- 英字は全て小文字で書く
- に単語以上を繋げるときは\_で繋ぐようにする
  例：　 order_list
